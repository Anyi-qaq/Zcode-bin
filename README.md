# ZCode-bin AUR Package

Arch Linux PKGBUILD for [ZCode](https://zcode.z.ai/) - AI-powered code editor by ZAI.

**Current version:** 3.1.8

## How It Works

ZCode publishes Linux `.deb` packages. This PKGBUILD extracts the `.deb`, uses only the app resources, and launches via [system Electron](https://wiki.archlinux.org/title/Electron) (`electron41`) — avoiding the ~250MB bundled Electron binary to reduce package size. The launcher script at `/usr/bin/zcode` handles Electron flags, Wayland support, and DE integration.

## Installation

### From AUR

```bash
yay -S zcode-bin
# or
paru -S zcode-bin
```

### Manual Build

```bash
git clone https://aur.archlinux.org/zcode-bin.git
cd zcode-bin
makepkg -si
```

## Dependencies

| Package | Purpose |
|---------|---------|
| `electron41` | System Electron runtime (avoids bundling ~250MB) |
| `python` | GLM AI agent scripts |
| `python-reportlab` | PDF generation in AI agent |
| `python-lxml` | XML processing in AI agent |
| `python-pillow` | Image processing in AI agent |
| `python-defusedxml` | Secure XML parsing in AI agent |
| `libstdc++` | C++ standard library for native modules |
| `nodejs` | Node.js runtime for editor tooling |
| `libgcc` | GCC runtime library |
| `ripgrep` | Code search (replaces bundled rg) |
| `asar` (makedepends) | Unpack/repack app.asar for path patching |

## Files

| File | Purpose |
|------|---------|
| `PKGBUILD` | Build script (the only file AUR needs beyond `.SRCINFO`) |
| `.SRCINFO` | AUR metadata (generated by `makepkg --printsrcinfo`) |
| `zcode-update-checker.sh` | Version checker, used by CI |
| `zcode.sh` | Launcher script template (installed to `/usr/bin/zcode`) |
| `.github/workflows/` | Daily cron: auto-check & push updates to AUR |

## Troubleshooting

### App Crashes on Startup

```bash
rm -rf ~/.cache/ZCode ~/.config/ZCode
zcode
```

### Electron Version Mismatch

If the launcher reports an Electron version mismatch during build, update `_electronversion` in `PKGBUILD` to match the detected version. The build script will print a warning with the correct value.
