# ZCode-bin AUR Package

Arch Linux PKGBUILD for [ZCode](https://zcode.z.ai/) - AI-powered code editor by ZAI.

**Current version:** 3.1.2

## How It Works

ZCode publishes Linux `.deb` packages. This PKGBUILD extracts the `.deb` and repackages it for Arch. Single source, zero build dependencies beyond base-devel.

## Installation

### From AUR

```bash
yay -S zcode-bin
# or
paru -S zcode-bin
```

### Manual Build

```bash
git clone https://aur.archlinux.org/zcode-bin.git
cd zcode-bin
makepkg -si
```

## Dependencies

| Package | Purpose |
|---------|---------|
| `gtk3` | GUI toolkit |
| `nss` | Network security |
| `libnotify` | Desktop notifications |
| `libxss` | Screen saver |
| `libxtst` | Input testing |
| `xdg-utils` | xdg-open / xdg-mime |
| `libsecret` | Keyring storage |
| `hicolor-icon-theme` | Icon theme |

## Files

| File | Purpose |
|------|---------|
| `PKGBUILD` | Build script (the only file AUR needs beyond `.SRCINFO`) |
| `.SRCINFO` | AUR metadata (generated by `makepkg --printsrcinfo`) |
| `zcode-update-checker.sh` | Version checker, used by CI |
| `.github/workflows/` | Daily cron: auto-check & push updates to AUR |

## Troubleshooting

### App Crashes on Startup

```bash
rm -rf ~/.cache/ZCode ~/.config/ZCode
zcode-bin
```
